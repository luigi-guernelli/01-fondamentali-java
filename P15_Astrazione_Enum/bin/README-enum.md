# Enum in Java: Costanti Nominative (P15)

## Descrizione
Enum per valori fissi con logica (costruttore + metodi).  
**Obiettivi**:  
1. Dichiarare enum con costanti.  
2. Aggiungere attributi/metodi (es. descrizione).  
3. Concreto: TipoForma per classificare oggetti Forma (da P14), switch enum.  

Sequenza: Dopo interface (P14), enum per parametrizzare astrazioni.

## Prerequisiti
- Interface (P14), switch (P09).
