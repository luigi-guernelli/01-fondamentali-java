# Getter e Setter in Java (Incampsulamento)

## Descrizione
Progetto per proteggere dati: Attributi 'private', accessibili solo via getter (lettura) e setter (scrittura controllata)
**Obiettivi**:
1. Incapsulamento: Nascondi attributi, evita modifiche errate.
2. Validazione: Setter controlla input es.(eta > 0).
3. Demo: Due oggetti Persona con modifiche + erroe.

Differenza da P04: Da public a private + metodi di accesso. Prepara per array di oggetti

## Prerequisiti
- OOP base (P03-P04: Costruttori, metodi).
