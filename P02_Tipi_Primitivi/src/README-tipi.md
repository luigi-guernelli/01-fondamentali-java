# Tipi primitivi e Variabili in Java

## Descrizione
Progetto base per comprendere i tipi di dati primitivi e String
**Obiettivi:**
1. Dichiarare e inizializzare variabili.
2. Tipi: 'int', 'double', 'boolean', 'char', 'String'
3. Output strutturato es.(profilo utente)

Differenza da Hello World: Focus su varabili "personali", non su saluto generico.

## Prerequisiti
- JDK 8+ installato

## Come Eseguire
1. Salva come 'Variabili.java'
2. Compila: 'javac Variabili.java'
3. Esegui: 'java Variabili'
