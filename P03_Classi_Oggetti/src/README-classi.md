# Classi e Oggetti in Java (Fondamenti OOP)

## Descrizione
Primo progetto OOP: Creazione di classi, oggetti, attribuiti e metodi speciali
**Obiettivi**
1. Definire una classe generica ('Persona').
2. Creare oggetti, assegnare valorri e chiamare metodi.
3. Introduzione a incapsulamento base (dati + azioni).

Differenza da precendenti: Da variabili "libere" a "oggetti strutturati" - prepara per array/ereditariet√†.

## Prerequisiti
- JDK 8+ e conoscenza di variabili/output.

## Come Eseguire
1. Salva i file in una cartella.  
2. Compila: `javac *.java`  
3. Esegui: `java Main`
