# ArrayList Dinamici in Java (P18)

## Descrizione
Liste dinamiche: Crescono/riduci, metodi utilità.  
**Obiettivi**:  
1. new ArrayList<>(); add(), remove(), get(), contains(), size().  
2. Iterazione come array (for/for-each).  
3. Concreto: ArrayList<Persona> (da P16), modifica lista dipendenti.  

Sequenza: Dopo array fissi (P17), ArrayList per flessibilità (no dimensione fissa).

## Prerequisiti
- Array/cicli (P17/P10), OOP (P16).
