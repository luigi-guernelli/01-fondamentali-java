# Title: Hello World in Java

## Description
Il classico progetto iniziale che introduce la sintassi Java e i tipi primitivi.

**Obiettivi**
1. Benvenuto nel mondo della programmazione.
2. Dichiarazione e usi di variabili con tipi: 'int', 'double', 'String, 'boolean', 'char'.
3. Output a console con 'System.out.println'.

## Prerequisiti
- JDK installato (Java Developer Kit, versione 8+/17+).
- IDE opzionale: IntelliJ IDEA Community o Eclipse.

## Come eseguire
1. Salva i file come 'Hello.java'.
2. Apri terminale nella cartella.
3. Compila: 'javac HelloWorld.java'
4. Esegui: 'java HelloWorld'
